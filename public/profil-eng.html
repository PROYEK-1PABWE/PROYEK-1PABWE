Graduate Profile

Graduate Profile

Graduates of bachelor of informatics have 10 graduate profiles that can be grouped into three groups. The first group is Software Engineer which includes graduate profiles: Mobile Application Developer, Web Developer (front-end, back-end, service), Full-Stack Developer, User Interface Designer, User Experience Designer, Software Analyst, Database Engineer, Software Quality Assurance dan Software Tester. The second group is Cybersecurity Practitioner, and the third group is AI (Artificial Inteligent) Engineer
